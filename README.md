# Game (2D)

>## 玩法介紹

    W:向上跳
    A:向左走
    D:向右走
    R:重新開始
    提示:遇到劍雨要躲避
    遊戲目標:找到寶箱

>## 腳本

    音效腳本:SoundManager
    玩家腳本:PlayerController
    劍雨腳本:weapon_controller、weapon_generator
    血量腳本:GameDirector
    金幣腳本:coin、ScoreTextScript
    畫面腳本:ClearDirector、LoseDirector
